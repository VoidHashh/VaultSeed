# Yahboom build package

This package is generated by GitHub Actions and is ready to flash to the Yahboom K210.

## Contents

- `kboot.kfpkg`: flash image for the device
- `firmware.bin`: raw firmware binary
- `ktool.py`: flashing tool already used by the repository
- `flash-yahboom.ps1` and `flash-yahboom.cmd`: Windows helpers
- `requirements-flash.txt`: Python dependency for flashing
- `BUILD-INFO.txt`: version, branch and commit
- `SHA256SUMS.txt`: checksums for the generated binaries

## Windows quick start

1. Find the serial port in `Device Manager -> Ports (COM & LPT)`.
2. The Yahboom usually appears as `USB-SERIAL CH340`.
3. If no COM port appears, install the CH340/341 driver and reconnect the board.
4. Install Python 3 if it is not already installed.
5. Install the flashing dependency:

```powershell
py -3 -m pip install -r requirements-flash.txt
```

6. Flash the device, replacing `COM6` with your port:

```cmd
flash-yahboom.cmd COM6
```

Or directly from PowerShell:

```powershell
powershell -ExecutionPolicy Bypass -File .\flash-yahboom.ps1 -Port COM6
```

7. Keep the board connected and powered on until the process finishes.

## Notes

- Default baud rate is `1500000`. If needed: `flash-yahboom.cmd COM6 921600`
- The Yahboom uses board mode `goE`
