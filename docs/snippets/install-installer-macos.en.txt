Open the `dmg` file and a new volume will be presented;
drag'n'drop the **VaultSeed Installer** to the **Applications** folder:

<img width="450" src="/VaultSeed/img/vaultseed-installer/dmg_volume.png" alt="vaultseed-installer's DMG volume" />

Before run, you'll need to remove the quarantine flag from application.
This occurs because we have not added the [Apple's code signing and notarization](https://support.apple.com/guide/security/app-code-signing-process-sec3ad8e6e53/web).

To fix this, open your terminal and execute the following command:

```bash
xattr -d com.apple.quarantine -r /Applications/vaultseed-installer.app
```
